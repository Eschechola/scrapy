# Pokemon Scrapy

<h4>Commands to run application</h4>

<br>

1.  Create virtual ENV

```bash
py -m venv env
```

<br>

2. Activate ENV

```bash
cd env
.\Scripts\activate
```

<br>

3. Install Packages

```bash
pip install scrapy
pip install pandas
```

<br>

4. Run Scrapy

```bash
scrapy runspider main.py
```
