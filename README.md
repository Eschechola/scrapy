# Pokemon Scrapy

<h4>Commands to run application</h4>

<br>

1.  Create virtual ENV

```bash
py -m venv env
```

<br>

2. Copy main.py to ENV


```bash
copy .\main.py env/
```

<br>

3. Activate ENV

```bash
cd env
.\Scripts\activate
```

<br>

4. Install Packages

```bash
pip install scrapy
pip install pandas
```

<br>

5. Run Scrapy

```bash
scrapy runspider main.py
```
